## Usage
Before generating, check this index to:
1. **Avoid repeating VILLAINS** in recent stories
2. **Avoid repeating SHOCK HOOKS** patterns
3. **Track KEY PHRASES** to ensure variety

## How to Update
After each generation, add a row with:
- **ID:** Folder name (timestamp)
- **Date:** YYYY-MM-DD
- **Topic:** Brief topic description
- **Villain:** Named villain used
- **Shock Hook:** First sentence pattern used
- **Key Phrases:** 3-5 unique phrases from the story
