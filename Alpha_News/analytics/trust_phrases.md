# 🤝 Trust Phrases

## Phrases That Build Credibility and Authority

Use these to establish trustworthiness before delivering the "pain injection".

---

## 📰 JOURNALISM/INVESTIGATION STYLE

### Source Attribution
```
როგორც ჩვენმა გამოძიებამ აჩვენა... (As our investigation showed...)
დოკუმენტების მიხედვით... (According to documents...)
შიდა წყაროების ინფორმაციით... (According to internal sources...)
გაჟონილი მასალების თანახმად... (According to leaked materials...)
ოფიციალური მონაცემებით... (According to official data...)
```

### Expert Citation
```
ექსპერტები აფრთხილებენ... (Experts warn...)
ანალიტიკოსების შეფასებით... (According to analysts...)
მკვლევარებმა დაადგინეს... (Researchers determined...)
უსაფრთხოების სპეციალისტები ამბობენ... (Security specialists say...)
```

### Verification Phrases
```
ეს დადასტურებული ფაქტია. (This is a confirmed fact.)
ჩვენ გადავამოწმეთ. (We verified this.)
ოფიციალური წყარო. (Official source.)
პირველადი მონაცემები. (Primary data.)
```

## 🎓 AUTHORITY BUILDERS

### Institutional References
```
Stanford-ის უნივერსიტეტის კვლევით... (According to Stanford research...)
MIT-ის ანალიზით... (According to MIT analysis...)
Harvard Business Review-ის მიხედვით... (According to HBR...)
The Economist-ი წერს... (The Economist writes...)
Reuters-ის ინფორმაციით... (According to Reuters...)
```

### Numerical Authority
```
73% მომხმარებლის... (73% of users...)
5-დან 4 კომპანია... (4 out of 5 companies...)
მილიარდი დოლარი... (billion dollars...)
მილიონობით ადამიანი... (millions of people...)
```

### Temporal Authority
```
ბოლო 10 წლის მონაცემებით... (According to last 10 years of data...)
2024 წლის ანგარიშით... (Per 2024 report...)
ისტორიაში პირველად... (For the first time in history...)
უკანასკნელი თვის განმავლობაში... (During the last month...)
```

## 🔒 INSIDER CREDIBILITY

### Exclusive Information
```
რასაც მედია არ გეტყვით... (What media won't tell you...)
კულისებში რა ხდება... (What happens behind the scenes...)
საიდუმლო, რომელიც ახლა გაჟონა... (Secret that just leaked...)
რაც მენეჯმენტმა დამალა... (What management hid...)
```

### Personal Authority
```
როგორც ინდუსტრიის შიგნიდან ვიცით... (As we know from inside the industry...)
წლების გამოცდილება გვაჩვენებს... (Years of experience show us...)
პირადად გადავამოწმეთ... (We personally verified...)
```

## ⚖️ BALANCE PHRASES (Before Attack)

### False Balance Setup
```
ოფიციალურად ამბობენ, რომ... მაგრამ რეალობა სხვაა.
(They officially say... but reality is different.)

კომპანია აცხადებს... თუმცა დოკუმენტები ცხადყოფს...
(Company declares... however documents reveal...)

PR განცხადება: "..." — რეალობა: "..."
(PR statement: "..." — Reality: "...")
```

---

## 📝 USAGE PATTERN

1. **START:** Trust phrase (establish authority)
2. **MIDDLE:** Deliver the bad news with evidence
3. **END:** Cynical conclusion

### Example Flow:
```
"როგორც ჩვენმა გამოძიებამ აჩვენა, [TRUST]
OpenAI-ის შიდა დოკუმენტები ადასტურებენ... [EVIDENCE]
ეს ნიშნავს, რომ შენი სამუშაო ადგილი საფრთხეშია. [PAIN]"
```

---

## ⚠️ RULES

1. Don't overuse - 1-2 trust phrases per post maximum
2. Place at beginning of key claims
3. Balance with emotional content
4. Never fabricate sources - use real or generic
