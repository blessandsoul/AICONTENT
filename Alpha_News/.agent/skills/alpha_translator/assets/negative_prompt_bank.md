## üî¥ CRITICAL (Always Include)

### Text/Watermark Issues
```
text error, typo, watermark, signature, messy text, illegible text, wrong text, misspelled text, extra text, random text, floating text
```

### Quality Issues
```
low res, low quality, jpeg artifacts, pixelated, blurry, grainy, noisy, compression artifacts, distorted
```

### Anatomy Issues
```
deformed hands, extra fingers, missing fingers, fused fingers, mutated hands, bad hands, malformed limbs, extra limbs, missing limbs
```

### Face Issues
```
human face on animal, deformed face, ugly face, duplicate face, extra eyes, missing eyes, asymmetric eyes
```

### Style Issues
```
2d, drawing, sketch, illustration, cartoon (when going for 3D), anime, manga, comic book style
```
