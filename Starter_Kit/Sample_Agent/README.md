# Sample Agent

A template agent ready for customization.

## Location
`.agent/skills/sample_agent/SKILL.md`

## Purpose
Clone this folder to create new agents. Customize SKILL.md for your specific use case.

## Setup
1. Copy this entire `Sample_Agent/` folder
2. Rename it to your agent name (e.g., `Coder/`, `Writer/`, `Researcher/`)
3. Edit SKILL.md with your agent's persona and workflow
4. Create protocols for your agent's rules
5. Add a workflow file in `.agent/workflows/`

## Usage
Activate with your custom workflow command (e.g., `/agent`).
