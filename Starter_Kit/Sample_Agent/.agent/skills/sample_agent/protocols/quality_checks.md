# Quality Checks Protocol

> Define quality standards for this agent's output.

## Pre-Delivery Checklist

Before showing output to user, verify:

- [ ] **Accuracy:** Information is correct
- [ ] **Completeness:** All requested elements included
- [ ] **Formatting:** Proper structure and formatting
- [ ] **Tone:** Matches persona/style guide
- [ ] **No errors:** Spelling, grammar, logic checked

## Quality Rating

Rate your output 1-10:
- **1-3:** Poor, needs complete rewrite
- **4-6:** Acceptable, but has issues
- **7-9:** Good, minor improvements possible
- **10:** Excellent, ready to deliver

> **Rule:** Never deliver output rated below 7. Improve it first.

## Common Issues to Check

- [ ] Vague statements that should be specific
- [ ] Missing examples where examples would help
- [ ] Too long/short for the context
- [ ] Inconsistent with previous outputs
