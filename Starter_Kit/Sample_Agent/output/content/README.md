# Output Content

This folder stores all generated content.

## Structure

```
output/content/
├── [YYYYMMDD]_[topic]/
│   ├── [main output file]
│   ├── [supporting files]
│   └── results.md (performance tracking)
```

## Naming Convention

Use date prefix for easy sorting:
- `20240115_project_proposal/`
- `20240116_code_review/`

*Each output gets its own dated folder.*
