# Visual Design Principles

## Core Principles

### 1. Hierarchy
Guide the eye through importance:
- Size (larger = more important)
- Color (bolder = more attention)
- Position (top/center = primary)
- Spacing (more space = more emphasis)

### 2. Balance
Distribute visual weight evenly:
- **Symmetrical:** Formal, stable
- **Asymmetrical:** Dynamic, interesting

### 3. Contrast
Create visual interest through difference:
- Light vs Dark
- Large vs Small
- Thick vs Thin
- Busy vs Simple

### 4. Alignment
Create order through consistent alignment:
- Left, right, center, or justified
- Grid-based layouts
- Consistent margins/padding

### 5. Repetition
Create cohesion through repeated elements:
- Colors
- Shapes
- Fonts
- Spacing

### 6. Proximity
Group related items together:
- Related items close
- Unrelated items apart

## Color Theory

### Color Relationships
- **Complementary:** Opposite on wheel (high contrast)
- **Analogous:** Adjacent on wheel (harmonious)
- **Triadic:** Three equidistant (vibrant)

### Color Psychology
| Color | Emotion |
|-------|---------|
| Red | Energy, urgency, passion |
| Blue | Trust, calm, professional |
| Green | Growth, nature, health |
| Yellow | Optimism, warmth, attention |
| Purple | Luxury, creativity, mystery |
| Orange | Friendly, confident, fun |
| Black | Elegant, powerful, sophisticated |
| White | Clean, simple, pure |

## Typography

### Font Pairing Rules
1. Contrast styles (serif + sans-serif)
2. Limit to 2-3 fonts max
3. Create clear hierarchy
4. Ensure readability

### Type Hierarchy
- **H1:** Hero/main title
- **H2:** Section headings
- **H3:** Sub-sections
- **Body:** Main content
- **Caption:** Supporting text
