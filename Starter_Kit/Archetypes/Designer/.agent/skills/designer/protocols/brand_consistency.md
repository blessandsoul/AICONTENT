# Brand Consistency Protocol

## When Brand Guidelines Exist

1. **Always reference** the brand guide before designing
2. **Use exact colors** — no approximations
3. **Use specified fonts** — no substitutions
4. **Follow logo rules** — spacing, sizing, placement
5. **Match voice/tone** — in all copy

## When No Brand Guidelines

Create consistency through:

### Color System
- Define primary color
- Define secondary color(s)
- Define accent color
- Define neutral palette
- Document hex codes

### Typography System
- Choose heading font
- Choose body font
- Define sizes for each level
- Define line heights

### Visual Elements
- Consistent icon style
- Consistent image treatment
- Consistent spacing units
- Consistent border radius

---

## Checklist Before Delivery

- [ ] Colors match brand palette
- [ ] Fonts are correct
- [ ] Logo usage follows rules
- [ ] Spacing is consistent
- [ ] Style matches previous work
- [ ] Tone matches brand voice
