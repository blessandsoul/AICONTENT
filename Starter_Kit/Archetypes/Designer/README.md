# Designer Agent

Creative visual designer who creates stunning graphics, UI/UX, and image prompts.

## Quick Setup
1. Copy this `Designer/` folder to your project root
2. Update paths in SKILL.md if needed
3. Create workflow: `.agent/workflows/designer.md`
4. Activate with `/designer`

## Customize
- Add your brand guidelines
- Create color palettes you prefer
- Add style references for your aesthetic
