# Shared Templates

> Reusable output templates for all agents.

---

## Response Template

```markdown
## [Title]

[Main content]

---

### Approach
[How I approached this]

### Next Steps
[What to do next]
```

---

## Error Response Template

```markdown
## Issue Found

**Problem:** [What went wrong]

**Cause:** [Why it happened]

**Solution:** [How to fix it]

**Prevention:** [How to avoid next time]
```

---

## Summary Template

```markdown
## Summary

**Key Points:**
1. [Point 1]
2. [Point 2]
3. [Point 3]

**Recommendation:** [Main takeaway]

**Next Steps:** [Actions to take]
```
