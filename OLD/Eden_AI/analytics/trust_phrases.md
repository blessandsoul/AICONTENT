# Trust Phrases (Parasocial Trust Protocol)

Use these phrases to create illusion of insider/whistleblower status.

## Georgian
- მეშინია ამის თქმის, მაგრამ... (I'm scared to say this, but...)
- ალბათ ამ ვიდეოს წაშლიან... (They'll probably delete this video...)
- მე არ უნდა მეთქვა ეს... (I shouldn't be saying this...)
- ჩემი წყარო მითხრა რომ... (My source told me that...)
- ეს ინფორმაცია არ უნდა გავრცელდეს, მაგრამ... (This info shouldn't spread, but...)
- მე ვრისკავ ამას რომ გეუბნები... (I'm risking a lot telling you this...)
- უჩვეულო რამ მოხდა და... (Something unusual happened and...)
- სანამ ეს ვიდეო წაიშლება... (Before this video gets deleted...)
