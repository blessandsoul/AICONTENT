# Inception Loop Protocol

## Purpose
Make the video rewatchable by creating a seamless loop.

## Rules
Block 7 ends so Block 1 sounds like CONTINUATION, not beginning.

## Planning Phase (REQUIRED)
1. **Choose END ACTION (Block 7):** "...и в этот момент система выбрала следующую жертву..."
2. **Choose START REVEAL (Block 1):** "[NAME] проснулся/узнал/получил..." (reveals WHO)
3. **Verify:** Does watching Block 7 → Block 1 feel like ONE continuous story?

## Connector Templates
- Georgian: "...ზუსტად იმ მომენტში, როდესაც სისტემამ თავისი შემდეგი მსხვერპლი აირჩია..."
- Russian: "...именно в тот момент, когда система выбрала следующую жертву..."

## Scoring
- Perfect seamless loop: +5 points
- Grammatically connects: +2 points
- No loop attempt: -3 points
